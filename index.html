ICDE (Integrated Cyber Defence Environment)

Project Overview:

The Integrated Cyber Defence Environment (ICDE) is a comprehensive, high-fidelity lab designed to simulate a modern Security Operations Center (SOC). It integrates various open-source security tools to provide a full-stack monitoring, detection, and response capability. This environment allows for hands-on experience in log analysis, incident response, threat simulation, and the development of automated response workflows using SOAR (Security Orchestration, Automation and Response) principles.

Key Objectives:

  • Simulate realistic cyber attack scenarios and practice defensive measures.
  • Gain proficiency in configuring and managing SIEM, NIDS, HIDS, and vulnerability assessment tools.
  • Develop and test incident response playbooks and automation scripts.
  • Understand log correlation across diverse systems for effective threat hunting.
  • Create a repeatable and extensible lab environment for continuous learning and experimentation.

Core Components & Technologies:

  • SIEM & HIDS: Wazuh for security information and event management, log analysis, and host-based intrusion detection.
  • NIDS: Suricata for network intrusion detection and prevention.
  • Vulnerability Management: OpenVAS for network vulnerability scanning and reporting.
  • Honeypot: Cowrie SSH/Telnet honeypot to capture attacker TTPs.
  • Log Aggregation & Analysis: Splunk (or ELK Stack) for centralized log collection and advanced analytics.
  • SOAR Platform: Shuffle for automating incident response workflows.
  • Target Environment: Windows (with Active Directory) and Linux virtual machines to simulate a typical corporate network.
  • Attack Simulation: Tools like Metasploit, Kali Linux utilities to generate attack traffic.

Learning Outcomes & Skills Gained:

  • Practical experience in setting up and managing a SOC-like environment.
  • Enhanced skills in log analysis, threat detection, and incident response.
  • Proficiency in using and integrating various cybersecurity tools (Wazuh, Suricata, Splunk, OpenVAS, Shuffle).
  • Understanding of attack vectors and defensive strategies in a simulated enterprise network.
  • Development of automation scripts for security operations.
